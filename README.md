# My dotfiles
========

Clone to `~/dotfiles`

```
ln -s ~/.vimrc ~/dotfiles/vim/.vimrc
ln -s ~/.gvimrc ~/dotfiles/vim/.gvimrc

mkdir ~/.vim
ln -s ~/.vim/after/ ~/dotfiles/vim/after/
ln -s ~/.vim/autoload/ ~/dotfiles/vim/autoload/
ln -s ~/.vim/bundle/ ~/dotfiles/vim/bundle/
ln -s ~/.vim/colors/ ~/dotfiles/vim/colors/
ln -s ~/.vim/ftplugin/ ~/dotfiles/vim/ftplugin/
ln -s ~/.vim/plugin/ ~/dotfiles/vim/plugin/
```
